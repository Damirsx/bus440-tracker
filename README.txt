🆘 ПОМОЩЬ И ПОДДЕРЖКА
═══════════════════════════════════════════════════════════════════════════════

❓ Сайт не открывается после деплоя
   → Подождите 5-10 минут (DNS обновление)
   → Проверьте что index.html в корне /public_html
   → Откройте консоль браузера (F12) - смотрите ошибки

❓ AI чат не работает
   → Убедитесь что backend запущен (локально или на хостинге)
   → Проверьте что порт 5000 открыт
   → Если backend не нужен - AI будет работать в базовом режиме

❓ Ошибка FTP при загрузке
   → Проверьте логин/пароль в ftp_config.txt
   → Проверьте что FTP_PATH правильный (обычно /public_html)
   → Попробуйте загрузить вручную через панель хостинга

❓ Как обновить сайт после изменений
   → Запустите AUTO_DEPLOY.bat снова
   → Или загрузите измененные файлы через FTP/панель управления

📊 ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ
═══════════════════════════════════════════════════════════════════════════════

Frontend:
   • Технология: HTML5 + Vanilla JavaScript + Tailwind CSS
   • Размер: ~150 KB
   • Зависимости: Только Tailwind CSS CDN (загружается автоматически)
   • Совместимость: Все современные браузеры
   • Адаптивность: Desktop, Tablet, Mobile

Backend (опционально):
   • Язык: Python 3.8+
   • Фреймворк: Flask
   • API: GigaChat by Sberbank
   • Зависимости: flask, flask-cors, requests (в requirements.txt)
   • Память: ~50 MB
   • CPU: Минимальная нагрузка

Требования к хостингу (минимальные):
   • Веб-сервер: Apache/Nginx
   • PHP: Не требуется
   • Python: 3.8+ (только для backend)
   • Место: 5 MB
   • База данных: Не требуется
   • SSL: Рекомендуется (обычно бесплатно)
